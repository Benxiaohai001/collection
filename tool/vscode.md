## 手动下载插件
* 通过web或其他方式下载安装包；
* 将安装包复制到vscode/bin目录下面；
* code  --install-extension codelldb-x86_64-windows.vsix 安装插件；\
出现如下内容则证明安装成功：\
```log
Extension 'codelldb-x86_64-windows.vsix' was successfully installed.
```
