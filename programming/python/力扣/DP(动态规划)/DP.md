# 动态规划
将一个复杂问题分解为多个相互关联的子问题，通过存储子问题的解，避免重复计算，来计算最终问题的解。
# 与递归问题有什么区别？
* 递归
    通过函数调用自身来解决问题的方法。问题通常可以分解为相同的子问题，关键在于定义基准情况和递归关系。
    * 优点
        * 代码简洁，逻辑清晰
        * 适用于树型结构问题
    * 缺点

* DP
    将问题可以分解为多个子问题，并存储子问题的解，避免重复计算。重叠子问题和最优子结构性质的问题。
    * 优点
        * 通过存储子问题的解，避免重复计算；
        * 重叠子问题和最优子结构问题
