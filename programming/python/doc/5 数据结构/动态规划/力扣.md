# [70. 爬楼梯](https://leetcode.cn/problems/climbing-stairs/description/)
1. [1]
2. [1+1, 2]
3. [1+1+1, 1+2, 2+1]
4. [1+1+1+1, 1+1+2, 1+2+1, 2+1+1, 2+2]
...
f(n) = f(n-1) + f(n-2)
```python
class Solution:
    def climbStairs(self, n: int) -> int:
        li = [1, 2] + [0]*n
        if n <= 2:
            return li[n-1]
        for i in range(2, n):
            li[i] = li[i-1] + li[i-2]
        return li[n-1]
```
