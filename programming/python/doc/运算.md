# 位运算
## 按位与（and） &
对应都为1时，结果为1，否则为0；
```python
>>> 1 & 1
1
>>> 1& 2
0
>>> 1& 3
1
>>>
```
## 按位或 （OR） |
对应只要有一个为1，结果为1，否则为0；
```python
>>> 1 | 1
1
>>> 1|2
3
>>> 1|3
3
```
## 按位异或 （XOR） ^
对应不同，结果为1，相同结果为0；
```python
>>> 1 ^ 1
0
>>> 1 ^ 2
3
>>> 1 ^ 3
2
>>>
```
## 按位取反 （NOT） ~
将每个位取反，1取0,0取1；
可以理解为-(x+1)
```python
>>> ~1
-2
>>> ~2
-3
>>> ~3
-4
>>> ~(-1)
0
>>> ~(-2)
1
>>> ~(-3)
2
>>>
```
## 左移 （） <<
将二进制的位向左移动指定位置A<<n
```python
>>> 1 << 1
2
>>> 2 << 1
4
>>> 3 << 1
6
>>>
```
## 右移 >>
将二进制的位向右移动指定位置A>>n
```python
>>> 1 >> 1
0
>>> 2 >> 1
1
>>> 3 >> 1
1
>>>
```
