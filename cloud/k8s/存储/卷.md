为pod中的容器提供了一种通过文件访问和[^共享数据]的方式。
# 用途
* config，secret配置文件
* 同一个pod中不同容器之间共享文件系统
* 两个不同pod之间共享文件系统（即使pod存在不同节点）
* 持久化存储数据
# 为什么卷很重要
* 数据持久性
* 共享存储
# 卷类型
* configmap
* downwardAPI \
为应用提供downward API
* emptyDir \
pod指定到某个节点此卷会创建。pod从节点删除时，卷也会删除
* hostpath \
hostpath 卷将主机节点文件系统上的文件或目录挂载到你的pod中。


[^共享数据]:共享数据可以发生在容器内不同本地进程之间，或不同容器之间，或多个pod之间。
